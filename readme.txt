Структура:

index - главная
users - "контроллер" пользователей, смотрит action и вызывает нужное из include
login - вход
logout - выход
style - парочка стилей
users.sql - дамп таблицы с тестовыми данными
include/ - подключаемые модули, прямой доступ закрыт .htaccess
	config - логин/хеш админа. Пароль XbJ7msxE
	mysql - настройки БД, подключение
	users_view - вывод пользователей
	user_*	- новый, правка, удалить, сохранить. Немного логики потом вызываем шаблон
templates/ - шаблоны, тоже закрыты .htaccess
	editform - форма редактирования/сохранения
	login - форма входа в админку
	main - главный шаблон, через него проходят все внутренние страницы

Проект выполнен в процедурном стиле, для базы использовал mysqli, проверял на PHP 5.6 и MySQL 5.7
Для начала работы нужно импортировать таблицу из дампа в базу данных, прописать в include/mysql.php логин/пароль для доступа к БД и название самой БД.
